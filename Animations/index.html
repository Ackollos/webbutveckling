This code makes a red square appear on the page that smoothly changes when you hover over it. When hovered, the box becomes bigger and turns orange.

HTML

        <div id="box">Makes the box bigger and orange</div>

CSS

        
        #box {
            width: 100px;
            height: 100px;
            background-color: red;
            transition: width 2s, height 2s, background-color 2s; 
          }
            
          #box:hover {
            width: 300px;
            height: 300px;
            background-color: orange;
          }
            
        #box-part{
              background-color: antiquewhite;
              border-radius: 15px;
              width: fit-content;
              margin-left: 30%;
        }

        #box {
            width: 100px;
            height: 100px;
            background-color: red;
            margin-left: 38%;
            transition: width 2s, height 2s, background-color 2s ; 
        }
              
        #box:hover {
            width: 300px;
            height: 300px;
            background-color: orange;
        }
Makes the box bigger and orange

This code creates a red circle on the page that moves back and forth using a CSS animation. The circle is 100 by 100 pixels, rounded to look like a circle, and starts moving after a short delay. The animation runs forever and moves the circle sideways across the screen and back.

HTML

        <div id="circle">Makes the circle move</div>

CSS

    @keyframes myAnimation {
        0%   {left:0px; top:0px;}
        25%  {left:350px; top:0px;}
        50%  {left:350px; top:0px;}
        75%  {left:0px; top:0px;}
        100% {left:0px; top:0px;}
    }

    #circle {
        width: 100px;
        height: 100px;
        background-color: red;
        position: absolute;
        animation-name: myAnimation;
        animation-duration: 4s;
        border-radius: 50px;
        margin-top: 10px;
        animation-delay: 1s;
        animation-iteration-count: infinite;
        margin-top: 70%;
    }

  
    #circle-part{
        background-color: antiquewhite;
        border-radius: 15px;
        width: fit-content;
        margin-left: 35%;
        padding: 0 5% 0 5% 0;
    }

    #circle {
        width: 100px;
        height: 100px;
        background-color: red;
        margin-left: 35%;
        position: relative;
        animation-name: myAnimation;
        animation-duration: 4s;
        border-radius: 50px;
        margin-top: 10px;
        animation-delay: 1s;
        animation-iteration-count: infinite;
        display: flex;
        justify-content: center;
        align-items: center;
        text-align: center;
    }
        
    @keyframes myAnimation {
        0%   {left:0px; top:0px;}
        25%  {left:400px; top:0px;}
        50%  {left:400px; top:0px;}
        75%  {left:0px; top:0px;}
        100% {left:0px; top:0px;}
    }
Makes the circle move
